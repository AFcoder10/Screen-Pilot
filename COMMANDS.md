## 🎯 Tips

1. **Natural Language**: Commands understand natural variations
   - "open calculator" = "open calc"
   - "write" = "type"
   - "solve" = "calculate"

2. **Math Operations**: Just type the math and it opens calculator
   - `65 + 2`
   - `100 * 25`
   - `200 / 4`

3. **Compound Commands**: Combine actions in one command
   - `open notepad and write your text here`
   - `open calc and solve 50 + 50`

4. **Window Management**: Control your windows
   - `maximize` - Make window fullscreen
   - `minimize` - Minimize current window
   - `close window` - Close active window
